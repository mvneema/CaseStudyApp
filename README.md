## üíª Local Installation

### Prerequisites
- Python 3.8+
- pip

### Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/case-study-solver.git
cd case-study-solver
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run case_study_solver.py
```

The app will open in your browser at `http://localhost:8501`
